---
title: 'Void a check'
api: 'POST /checks/{checkId}/void'
---

Void a check using CheckId

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="checkId" type="string" required>
  Path parameter `checkId`.
</ParamField>

## Request

Endpoint: `POST /checks/{checkId}/void`

### Example

```bash
curl --location "$BASE_URL/checks/{checkId}/void" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```

## Response example

```json
{
  "success": true,
  "message": "Successfully Voided"
}
```
