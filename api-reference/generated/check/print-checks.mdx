---
title: 'Print  checks'
api: 'POST /checks/print'
---

You can print checks by passing check id and paper type,if you would like to print multiple checks you need to pass checkId as array.

Payload parameters

 
  paperType
  Required    
        Integer
        Must be an integer from Available paper types
  
 
  checkId
  Required    
        Array
        An Array of check id to print
  



Available paper types





Paper type
Description



1
Check on Top



2
Check on Middle



3
Check on Bottom



4
Three per page



5
Wallet size



6
Simple print




## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Body parameters

<ParamField body="paperType" type="number">
  Request body field `paperType`.
</ParamField>

<ParamField body="checkId" type="string[]">
  Request body field `checkId`.
</ParamField>

## Request

Endpoint: `POST /checks/print`

### Example

```bash
curl --location "$BASE_URL/checks/print" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
