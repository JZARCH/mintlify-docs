---
title: 'Update a check'
api: 'PUT /checks/{checkId}'
authMethod: 'bearer'
---

Update details of an existing check

Path parameters

 
  checkId
  Required    
         The id of the check to update.
  



Payload parameters

 
  bankAccountId
    Required 
         String 
        Must be a bank account ID
  
    



  payeeId
   Required    
       String 
   Must be a payee ID
      Optional if noPayee is true, otherwise required.
   
  



  categoryId
   Optional 
      String 
   Must be a category ID
  


  serialNumber
   Optional 
    String 
    By default increment every time a check is created, If a serialNumber is passed, that will override any default.
  



  issueDate
   Optional 
    String yyyy-mm-dd 
 By default it will be current date, If a issueDate is passed, that will 
    override any default.
  



  amount
  Required  
    Decimal 
    Optional if noAmount  is  true, otherwise required.
  



  memo
   Optional 
    String 
   Max of 150 characters to be included on the memo line of the check.
  



  note
   Optional 
     String 
     An internal note that identifies this check. Must be no longer than 150 characters.
  



  accountNumber
   Optional 
    String 
    Any account number associated with this check
  



  invoiceNumber
   Optional  
    String 
    Any invoice number associated with this check
  



  noSign
   Optional 
    Boolean 
   if true ,check will create without signature ,By default false
  



  noAmount
   Optional  
    Boolean 
    iftrue ,check will create without amount,By default false
  




  noDate
   Optional 
     Boolean 
     if true ,check will create without date,By default false  
  



  noPayee
   Optional 
    Boolean 
    if true ,check will create without payee,By default false
