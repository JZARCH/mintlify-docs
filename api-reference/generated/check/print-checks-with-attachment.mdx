---
title: 'Print  checks with attachment'
api: 'POST /checks/print-with-attachment'
authMethod: 'bearer'
---

You can print checks by passing check id and paper type,if you would like to print multiple checks you need to pass checkId as array.

**Payload parameters**

paperTypeRequiredIntegerMust be an integer from Available paper typescheckIdRequiredArrayAn Array of check id to printasyncOptionalInteger

_Available_ `_paper types_`

Paper typeDescription1Check on Top2Check on Middle3Check on Bottom4Three per page5Wallet size6Simple print

## Body parameters

<ParamField body="paperType" type="number">
  Request body field `paperType`.
</ParamField>

<ParamField body="checkId" type="string[]">
  Request body field `checkId`.
</ParamField>

## Request

Endpoint: `POST /checks/print-with-attachment`

### Example

```bash
curl --location "$BASE_URL/checks/print-with-attachment" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
