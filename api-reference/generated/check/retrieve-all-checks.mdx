---
title: 'Retrieve all checks'
api: 'GET /checks'
authMethod: 'bearer'
---

List all check records

**Query parameters**

perPageNumber of records that you want to receive in the response,By default 10 records will belistedpageFor navigating through pages ,By default first page will be listedtermFilter records by using any keywordsstatusFilter records by using check status ,Eg:new , void ,mailed ,refunded , emailed , eprinted ,printedreferenceIdFilter records by referenceIdcolumnThe column name to be used for filtering records. Supported columns include payee_name,payee_nick_name,serial_number,amount,bank_account_account_name,bank_account_nick_name,memo,noteSearchTypeDefines how the search is applied to the chosen column. Eg- exact or like

## Query parameters

<ParamField query="perPage" type="string">
  Number of records that you want to receive in the response,By default 10 records will be listed
</ParamField>

<ParamField query="page" type="string">
  By default first page will be listed
</ParamField>

<ParamField query="status" type="string">
  Filter record by using check status 
</ParamField>

<ParamField query="term" type="string">
  Filter records by using any keywords
</ParamField>

<ParamField query="referenceId" type="string">
  Query parameter `referenceId`.
</ParamField>

  Request body field `data.checks`.
</ParamField>

## Request

Endpoint: `GET /checks`

### Example

```bash
curl --location "$BASE_URL/checks" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
