---
title: 'Retrieve a check'
api: 'GET /checks/{checkId}'
---

Details of an existing check.

Path parameters

 
  checkId
  Required    
         The id of the check to retrieved.
  



## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="checkId" type="string" required>
  Path parameter `checkId`.
</ParamField>

## Request

Endpoint: `GET /checks/{checkId}`

### Example

```bash
curl --location "$BASE_URL/checks/{checkId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
