---
title: 'Retrieve all attachments details'
api: 'GET /checks/{checkId}/attachments'
---

Retrieve all attachments of a existing check.

Path parameters

 
  checkId
  Required    
         The id of the check.
  



## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="checkId" type="string" required>
  Path parameter `checkId`.
</ParamField>

## Request

Endpoint: `GET /checks/{checkId}/attachments`

### Example

```bash
curl --location "$BASE_URL/checks/{checkId}/attachments" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
