---
title: 'Send Virtual Card'
api: 'POST /send-payment/virtualCard'
---

Send a Virtual Card to your payee.  
Allowed sourceType : `wallet` , `bankAccount`

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Body parameters

<ParamField body="sourceType" type="string">
  Request body field `sourceType`.
</ParamField>

<ParamField body="sourceId" type="string">
  Request body field `sourceId`.
</ParamField>

<ParamField body="amount" type="string">
  Request body field `amount`.
</ParamField>

<ParamField body="description" type="string">
  Request body field `description`.
</ParamField>

<ParamField body="payeeId" type="string">
  Request body field `payeeId`.
</ParamField>

<ParamField body="referenceNo" type="string">
  Request body field `referenceNo`.
</ParamField>

## Request

Endpoint: `POST /send-payment/virtualCard`

### Example

```bash
curl --location "$BASE_URL/send-payment/virtualCard" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
