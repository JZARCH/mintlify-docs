---
title: 'List All Virtual Card Sent'
api: 'GET /send-payment/virtualCard'
---

Get all the Virtual Cards you have sent.

**Query parameters**

referenceNoFilter records by referenceNo

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Request

Endpoint: `GET /send-payment/virtualCard`

### Example

```bash
curl --location "$BASE_URL/send-payment/virtualCard" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
