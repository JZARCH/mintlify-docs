---
title: 'List All ACH Payments'
api: 'GET /send-payment/ach'
---

Get all the ACH payments you have initiated.

**Query parameters**

referenceNoFilter records by referenceNo

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Request

Endpoint: `GET /send-payment/ach`

### Example

```bash
curl --location "$BASE_URL/send-payment/ach" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
