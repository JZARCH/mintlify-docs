---
title: 'Send Wire Payment'
api: 'POST /send-payment/wire'
authMethod: 'bearer'
---

Initiate an Wire Payment from a wallet or bank account.  
Allowed sourceType : `wallet` ,`bankAccount`

## Body parameters

<ParamField body="sourceType" type="string">
  Request body field `sourceType`.
</ParamField>

<ParamField body="sourceId" type="string">
  Request body field `sourceId`.
</ParamField>

<ParamField body="amount" type="string">
  Request body field `amount`.
</ParamField>

<ParamField body="description" type="string">
  Request body field `description`.
</ParamField>

<ParamField body="payeeId" type="string">
  Request body field `payeeId`.
</ParamField>

<ParamField body="payeeBankAccount" type="string">
  Request body field `payeeBankAccount`.
</ParamField>

<ParamField body="referenceNo" type="string">
  Request body field `referenceNo`.
</ParamField>

## Request

Endpoint: `POST /send-payment/wire`

### Example

```bash
curl --location "$BASE_URL/send-payment/wire" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
