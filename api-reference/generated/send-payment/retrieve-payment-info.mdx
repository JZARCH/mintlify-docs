---
title: 'Retrieve Payment Info'
api: 'GET /send-payment/{paymentId}/info'
---

You can retrieve the payment status by passing the payment ID.

**PAYMENT STATUS**

| **Status** | **Description** |
| --- | --- |
| PENDING | The payment request has been created but not initiated yet. |
| PROCESSING | The payment has been initiated and is in progress. |
| COMPLETED | The payment has been successfully executed and finalized. |
| FAILED | Payment was rejected or returned |

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="paymentId" type="string" required>
  Path parameter `paymentId`.
</ParamField>

## Request

Endpoint: `GET /send-payment/{paymentId}/info`

### Example

```bash
curl --location "$BASE_URL/send-payment/{paymentId}/info" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
