---
title: 'List All Check Mail Payments'
api: 'GET /send-payment/checkMail'
---

Get all the Check Mail payments you have initiated.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Request

Endpoint: `GET /send-payment/checkMail`

### Example

```bash
curl --location "$BASE_URL/send-payment/checkMail" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
