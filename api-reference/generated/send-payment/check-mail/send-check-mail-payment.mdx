### Send Payment CheckMail

This API endpoint allows you to initiate a check mailing payment directly from a specified wallet.

**Payload Parameter**

| Key | Type | Required | Description |
| --- | --- | --- | --- |
| sourceType | String | ✅ Yes | Type of payment source (e.g., "wallet"). |
| sourceId | String | ✅ Yes | ID of the wallet |
| amount | String | ✅ Yes | Amount to initiate check mailing. |
| description | String | ✅ Yes | Description for the transaction |
| payeeId | String | ✅ Yes | ID of the recipient/payee to whom the amount will be sent. |
| referenceNo | String | ⬜ Optional | Custom reference number for tracking or identifying the transaction. |
| shippingType | String | ✅ Yes | Type of mailing service used (e.g., USPS_PRIORITY_MAIL, FEDEX_NEXT_DAY_PM). |
| attachmentIds | array | ⬜ Optional | List of uploaded attachment IDs |

**Available Shipping Types**

| Shipping Type | Shipping Type Code |
| --- | --- |
| USPS First Class Mail | USPS_FIRST_CLASS_MAIL |
| USPS First Class With Tracking | USPS_FIRST_CLASS_WITH_TRACKING |
| USPS Express Mail | USPS_EXPRESS_MAIL |
| USPS Priority Mail | USPS_PRIORITY_MAIL |
| FedEx Next Day PM | FEDEX_NEXT_DAY_PM |
