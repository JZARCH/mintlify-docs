---
title: 'Add Attachment'
api: 'POST /attachment'
authMethod: 'bearer'
---

This API uploads an attachment file (example: PDF,document) and returns a unique attachment ID.

Supported File Types

| Key | Type | Required | Description |
| --- | --- | --- | --- |
| image | file | Optional | Supported file formats: pdf, xls, xlsx, rtf, doc, docx, txt, jpg, png. |
| fileUrl | Text | Optional | Public URL of the file to upload or process. Supported file formats: pdf, xls, xlsx, rtf, doc, docx, txt, jpg, png. |

## Body parameters

<ParamField body="success" type="boolean">
  Request body field `success`.
</ParamField>

<ParamField body="data" type="object">
  Request body field `data`.
</ParamField>

<ParamField body="data.id" type="string">
  Request body field `data.id`.
</ParamField>

## Request

Endpoint: `POST /attachment`

### Example

```bash
curl --location "$BASE_URL/attachment" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```

## Response example

```json
{
  "success": true,
  "data": {
    "id": "OYxZqXGKdeRzVvJ"
  }
}
```
