---
title: 'Delete Attachment'
api: 'DELETE /attachment/{attachment_id}'
authMethod: 'bearer'
---

This API deletes an uploaded attachment using the provided `attachment_id`.Once deleted, the file cannot be accessed or retrieved again.

## Path parameters

<ParamField path="attachment_id" type="string" required>
  Path parameter `attachment_id`.
</ParamField>

## Body parameters

<ParamField body="success" type="boolean">
  Request body field `success`.
</ParamField>

## Request

Endpoint: `DELETE /attachment/{attachment_id}`

### Example

```bash
curl --location "$BASE_URL/attachment/{attachment_id}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```

## Response example

```json
{
  "success": true
}
```
