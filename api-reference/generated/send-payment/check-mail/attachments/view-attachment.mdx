---
title: 'View Attachment'
api: 'GET /attachment/{attachment_id}'
authMethod: 'bearer'
---

This API retrieves the uploaded attachment using the provided `attachment_id` and returns a **secure pre-signed URL** to view or download the file.

## Path parameters

<ParamField path="attachment_id" type="string" required>
  Path parameter `attachment_id`.
</ParamField>

## Request

Endpoint: `GET /attachment/{attachment_id}`

### Example

```bash
curl --location "$BASE_URL/attachment/{attachment_id}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
