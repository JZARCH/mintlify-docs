## check.print.async (success)

Triggered when an **asynchronous check print job completes successfully**. The PDF is ready to download from the provided URL.

**Trigger:** `AsynchronousCheckPrintJob` finishes generating the print PDF for one or more checks.

**Source:** `AsynchronousCheckPrintJob::handle()` → `postOcwUsersWebhook()`

---

## Payload Field Reference

| Field | Type | Description |
|-------|------|-------------|
| `data.status` | string | `"success"` — print job completed |
| `data.requestId` | string (UUID) | The async print request ID — matches the request that triggered printing |
| `data.url` | string | Pre-signed S3 URL to the generated check PDF — download promptly (expires) |
| `data.checkIds` | array of strings | Encrypted IDs of the checks included in this print batch |

> **Note:** The `data.url` is a pre-signed S3 URL. Download the PDF as soon as possible after receiving this event.
