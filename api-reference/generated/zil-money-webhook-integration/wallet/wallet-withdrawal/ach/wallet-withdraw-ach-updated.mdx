## wallet.withdraw.ach.updated

Triggered when the status of a **wallet ACH withdrawal** changes.

**Trigger:** The withdrawal transfer status is updated by the banking network.

**Source:** `BankingAchTransferRequestObserver::updated()` â€” fires only when `status` field changes.

---

Same schema as `wallet.withdraw.ach.initiated`. The `data.status` and `data.updatedAt` fields reflect the new state.
