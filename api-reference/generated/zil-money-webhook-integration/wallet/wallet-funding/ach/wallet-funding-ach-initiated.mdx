## wallet.funding.ach.initiated

Triggered when a user initiates an **ACH transfer to fund their Zil Money wallet** from a linked bank account.

**Trigger:** User adds funds to their Zil Money wallet via ACH bank pull.

**Source:** `BankingAchTransferRequestObserver::created()` → `BankingAchWalletWithdrawalTransferRequestResource`

---

## Payload Field Reference

| Field | Type | Description |
|-------|------|-------------|
| `data.id` | string (UUID) | Unique identifier for this ACH transfer |
| `data.fromWallet` | string\|null | Source wallet (null for funding — funds come from bank) |
| `data.toWallet` | string | Destination wallet name (e.g., `"primary"`) |
| `data.bankAccountId` | string | Encrypted ID of the linked bank account being debited |
| `data.amount` | string | Amount being transferred (2 decimal places) |
| `data.description` | string | Transfer description |
| `data.status` | string | `PENDING`, `PROCESSING`, `COMPLETED`, `CANCELED`, `FAILED`, `RETURNED` |
| `data.updatedAt` | ISO 8601 datetime | Last updated timestamp |
| `data.createdAt` | ISO 8601 datetime | Creation timestamp |
