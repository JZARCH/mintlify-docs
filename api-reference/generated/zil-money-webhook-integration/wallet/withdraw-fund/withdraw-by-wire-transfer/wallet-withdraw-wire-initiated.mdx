## wallet.withdraw.wire.initiated

Triggered when a user initiates a **wire transfer withdrawal from their Zil Money wallet** to a linked bank account.

**Trigger:** User submits a wire withdrawal from their wallet.

---

## Payload Field Reference

| Field | Type | Description |
|-------|------|-------------|
| `data.id` | string (UUID) | Unique identifier for this wire withdrawal |
| `data.fromWallet` | string (UUID) | Source wallet UUID |
| `data.toBankAccount` | string | Encrypted destination bank account ID |
| `data.amount` | string | Withdrawal amount (2 decimal places) |
| `data.description` | string | Transfer description |
| `data.status` | string | `PENDING`, `PROCESSING`, `COMPLETED`, `CANCELED`, `FAILED`, `RETURNED` |
| `data.updatedAt` | ISO 8601 datetime | Last updated timestamp |
| `data.createdAt` | ISO 8601 datetime | Creation timestamp |

## HEADERS

| Header | Value |
| --- | --- |
| Content-Type | application/json |
| Signature | {{hmac_signature}}<br/>HMAC-SHA256 of the raw JSON request body, signed with your webhook_secret: hash_hmac('sha256', rawJsonBody, webhookSecret) |

## Example Request

```http
POST /webhooks/zilmoney HTTP/1.1
Host: your-server.com
Content-Type: application/json
Signature: {{hmac_signature}}
```
