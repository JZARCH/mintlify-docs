## wallet.withdraw.ach.updated

Triggered when the status of a **wallet ACH withdrawal** changes.

**Trigger:** The withdrawal transfer status is updated by the banking network.


---

Same schema as `wallet.withdraw.ach.initiated`. The `data.status` and `data.updatedAt` fields reflect the new state.

## HEADERS

| Header | Value |
| --- | --- |
| Content-Type | application/json |
| Signature | {{hmac_signature}}<br/>HMAC-SHA256 of the raw JSON request body, signed with your webhook_secret: hash_hmac('sha256', rawJsonBody, webhookSecret) |

## Example Request

```http
POST /webhooks/zilmoney HTTP/1.1
Host: your-server.com
Content-Type: application/json
Signature: {{hmac_signature}}
```
