## card-program.card.created

Triggered when a new **card program issuing record** (virtual or physical card) is successfully created for a user.

**Trigger:** A `CardProgramIssuingRecord` is created and the card is issued via the card network.


---

## Payload Field Reference

| Field | Type | Description |
|-------|------|-------------|
| `data.id` | string (UUID) | Unique identifier for the card issuing record |
| `data.walletId` | string (UUID) | UUID of the wallet associated with this card |
| `data.status` | string | Issuing record status: `PENDING` at creation, then `ACTIVE`, `INACTIVE`, `BLOCKED`, `CANCELED` |
| `data.card.id` | string (UUID) | UUID of the card detail record |
| `data.card.cardProgramIssuingRecordId` | string (UUID) | UUID of the parent issuing record |
| `data.card.cardType` | string | `virtual` or `physical` (lowercase) |
| `data.card.label` | string | User-defined card label (often the cardholder name) |
| `data.card.nameOnCard` | string | Name printed on the card |
| `data.card.limitAmount` | string | Spending limit amount |
| `data.card.limitInterval` | string | Limit interval: `daily`, `weekly`, `monthly`, `yearly`, `lifetime`, `per_authorization` (lowercase) |
| `data.card.last4` | string | Last 4 digits of the card number |
| `data.card.cardStatus` | string | Card network status: `active`, `inactive`, `blocked`, `canceled` (lowercase) |
| `data.card.expiryMonth` | string | Card expiry month (no leading zero, e.g., `"5"`) |
| `data.card.expiryYear` | string | Card expiry year (4 digits) |
| `data.card.cardIssueType` | string | Issuance type: `WALLET_CARD`, `EMPLOYEE`, `VENDOR`, `PERSONAL`, `RECHARGEABLE_GIFT_CARD` |
| `data.card.wallet` | object\|null | Associated wallet: `id`, `availableBalance`, `currentBalance`, `walletName` (no `walletType` in V3) |
| `data.card.restrictionSettings` | object | Spending restrictions (empty object `{}` when none configured; see structure below) |
| `data.updatedAt` | ISO 8601 datetime | Last updated timestamp |
| `data.createdAt` | ISO 8601 datetime | Creation timestamp |

## restrictionSettings Object

| Field | Type | Description |
|-------|------|-------------|
| `validTill` | string (date) | Card expiry/validity date in `YYYY-MM-DD` format |
| `allowedMerchantNames` | array of strings | Whitelisted merchant names (lowercase) |
| `allowedMerchantCategoryCode` | array of strings | Whitelisted MCC codes |
| `allowedMerchantCountry` | array of strings | Whitelisted 2-letter country codes |
| `allowedMerchantCity` | array of strings | Whitelisted city names |
| `allowedTime.timezone` | string | IANA timezone (e.g., `"America/Chicago"`) |
| `allowedTime.times` | array of objects | Allowed time windows: `[{ "from": "HH:MM", "to": "HH:MM" }]` |

## HEADERS

| Header | Value |
| --- | --- |
| Content-Type | application/json |
| Signature | {{hmac_signature}}<br/>HMAC-SHA256 of the raw JSON request body, signed with your webhook_secret: hash_hmac('sha256', rawJsonBody, webhookSecret) |

## Example Request

```http
POST /webhooks/zilmoney HTTP/1.1
Host: your-server.com
Content-Type: application/json
Signature: {{hmac_signature}}
```
