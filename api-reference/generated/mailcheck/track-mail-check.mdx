---
title: 'Track Mail Check'
api: 'GET /mailchecks/{checkId}/tracking'
authMethod: 'bearer'
---

**Response Data**

carrierCarrier name (USPS,Fedex)trackingCodeTracking code numberstatusCurrent status of the tracking ('pre_transit','in_transit','delivered','error','unknown','out_for_delivery','return_to_sender' or 'cancelled')estimatedDeliveryDateEstimated delivery datetrackingDetailsTracking updates details

## Request

Endpoint: `GET /mailchecks/{checkId}/tracking`

### Example

```bash
curl --location "$BASE_URL/mailchecks/{checkId}/tracking" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
