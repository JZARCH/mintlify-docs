---
title: 'Retrieve all mail options'
api: 'GET /mailchecks/mailingOptions'
---

Retrieve all available mail options,including shipping types,Paper types and Inform types,You can use this data to create mail check.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Request

Endpoint: `GET /mailchecks/mailingOptions`

### Example

```bash
curl --location "$BASE_URL/mailchecks/mailingOptions" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
