This API allows you to upload PDF file as attachment,
There are two types of uploads you can perform:
Upload via remote URL with, (fileUrl)
You can provide any remote public accessible url as ’fileUrl’ parameter , this file will be fetched and will be added as attachment
	
Simple Upload (attachment), 		
You can directly upload the file via post method

Path parameters

 
  checkId
  Required    
         Id of the Check.
  



Payload parameters

 
  attachment
  Required    
        File
        File of type pdf,Optional if fileUrl is passed
  

 
  fileUrl
  Required    
        String
        Must be a public accessible url ,Optional if attachment is passed
