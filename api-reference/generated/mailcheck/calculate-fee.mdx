---
title: 'Calculate Fee'
api: 'POST /mailchecks/calculateFee'

---

You can calculate check mail fees by using same mail creation payload parameters.

## Body parameters

<ParamField body="isShippingToCustomAddress" type="boolean">
  Request body field `isShippingToCustomAddress`.
</ParamField>

<ParamField body="customFromAddressId" type="string">
  Request body field `customFromAddressId`.
</ParamField>

<ParamField body="customToAddressId" type="string">
  Request body field `customToAddressId`.
</ParamField>

<ParamField body="customShippingTypeId" type="number">
  Request body field `customShippingTypeId`.
</ParamField>

<ParamField body="mailChecks" type="object[]">
  Request body field `mailChecks`.
</ParamField>

## Request

Endpoint: `POST /mailchecks/calculateFee`

### Example

```bash
curl --location "$BASE_URL/mailchecks/calculateFee" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
