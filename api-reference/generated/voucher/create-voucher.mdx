Create New Voucher

Payload parameters

 
  voucherNumber
  Required  
       String  
         Must be no longer than 50 characters.
  
 
 
  memo
  Optional  
       String  
          Must be no longer than 100 characters.
  
 
 
  date
  Optional  
       String yyyy-mm-dd  
         By default it will be voucher creation date, If a date is passed, that will
             override any default.
  
 

  payeeId
  Required  
       String  
         Must be a payee ID
  
 

  voucherItems
  Optional  
       Array  
         An array of voucherItems attributes
  
 



voucherItems attributes


 
  invoiceNumber
  Required  
       String  
         Must be no longer than 50 characters.
  
 
 
  name
  Optional  
       String  
          Must be no longer than 100 characters.
  
 
 
  description
  Optional  
       String   
          Must be no longer than 100 characters.
  
 

  quantity
  Optional  
       Integer  
         
  
 

  unitCost
  Optional  
       Decimal  
         
  
 

  total
  Required  
       Decimal
