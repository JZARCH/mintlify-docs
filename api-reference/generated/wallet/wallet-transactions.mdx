List all transactions of a wallet.

</ParamField>

## Request

Endpoint: `GET /wallet/{uuid}/transactions`

### Example

```bash
curl --location "$BASE_URL/wallet/{uuid}/transactions" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
