---
title: 'Wallet Internal Transfer'
api: 'POST /wallet/internal-transfer'
authMethod: 'bearer'
---

You can transfer funds from one wallet to another within the same account using this endpoint.

## Body parameters

<ParamField body="fromWallet" type="string">
  Request body field `fromWallet`.
</ParamField>

<ParamField body="toWallet" type="string">
  Request body field `toWallet`.
</ParamField>

<ParamField body="amount" type="string">
  Request body field `amount`.
</ParamField>

<ParamField body="description" type="string">
  Request body field `description`.
</ParamField>

## Request

Endpoint: `POST /wallet/internal-transfer`

### Example

```bash
curl --location "$BASE_URL/wallet/internal-transfer" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
