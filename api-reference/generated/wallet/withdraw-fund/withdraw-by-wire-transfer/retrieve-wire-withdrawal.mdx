## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="wireId" type="string" required>
  Path parameter `wireId`.
</ParamField>

## Request

Endpoint: `GET /wallet/withdraw/wire/{wireId}`

### Example

```bash
curl --location "$BASE_URL/wallet/withdraw/wire/{wireId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
