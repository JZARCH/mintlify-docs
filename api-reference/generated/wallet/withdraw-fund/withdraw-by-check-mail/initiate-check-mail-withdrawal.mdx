---
title: 'Initiate Check Mail Withdrawal'
api: 'POST /wallet/withdraw/mail/initiate'
authMethod: 'bearer'
---

**Payload Parameter**

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| fromWallet | string | ✅ Yes | ID of the wallet from which funds will be withdrawn. |
| amount | string | ✅ Yes | Amount to withdraw |
| description | string | ✅ Yes | Description for the withdrawal transaction. |
| customFromAddressId | string | ✅ Yes | Custom address ID from which the check will be mailed. |
| customToAddressId | string | ✅ Yes | Custom recipient address ID to which the check will be sent. |
| shippingType | string | ✅ Yes | Type of mailing service used (e.g., FEDEX_NEXT_DAY_PM, USPS_FIRST_CLASS_MAIL). |

**Available Shipping Types**

| **Shipping Type** | **Shipping Type Code** |
| --- | --- |
| USPS First Class Mail | USPS_FIRST_CLASS_MAIL |
| USPS First Class With Tracking | USPS_FIRST_CLASS_WITH_TRACKING |
| USPS Express Mail | USPS_EXPRESS_MAIL |
| USPS Priority Mail | USPS_PRIORITY_MAIL |
| FedEx Next Day PM | FEDEX_NEXT_DAY_PM |
