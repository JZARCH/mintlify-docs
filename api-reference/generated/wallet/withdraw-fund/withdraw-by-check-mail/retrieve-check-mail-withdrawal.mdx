## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="mailId" type="string" required>
  Path parameter `mailId`.
</ParamField>

## Request

Endpoint: `GET /wallet/withdraw/mail/{mailId}`

### Example

```bash
curl --location "$BASE_URL/wallet/withdraw/mail/{mailId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
