---
title: 'Retrieve ACH wallet funding'
api: 'GET /wallet/fund-wallet/{walletId}'
authMethod: 'bearer'
---

_**STATUS AND DESCRIPTIONS**_

- **PENDING**: Once the fund wallet request is created, the status will be PENDING.
    
- **PROCESSING**: Once the request is created, the transaction will start to process, and the status will change to PROCESSING instantly.
    
- **COMPLETED**: Once the transaction process is successfully completed, the funds will be available in the wallet.
    
- **FAILED**: When the transaction is rejected, failed, or returned during processing, the status will be FAILED.
    
- **RETURNED**: After the funds are released, if the transaction is returned as an ACH return, the amount will be deducted from the wallet at this time.
