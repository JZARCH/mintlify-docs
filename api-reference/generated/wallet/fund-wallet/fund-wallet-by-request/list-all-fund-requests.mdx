title: 'List All Fund Requests'
api: 'GET /wallet/fund/requests'
---

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Request

Endpoint: `GET /wallet/fund/requests`

### Example

```bash
curl --location "$BASE_URL/wallet/fund/requests" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
