---
title: 'Show Wallet Gift Card Transaction Details'
api: 'GET /card-program/{cardProgramId}/transactions'
authMethod: 'bearer'

This API endpoint retrieves the transaction history and details for a specific gift card linked to a wallet using the card ID.

</ParamField>

## Request

Endpoint: `GET /card-program/{cardProgramId}/transactions`

### Example

```bash
curl --location "$BASE_URL/card-program/{cardProgramId}/transactions" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
```
