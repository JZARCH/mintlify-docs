---
title: 'Retrieve Wallet Gift Card'
api: 'GET /card-program/gift-card/{giftCardId}'
---

Retrieves the details of a specific gift card using its unique ID.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="giftCardId" type="string" required>
  Path parameter `giftCardId`.
</ParamField>

## Request

Endpoint: `GET /card-program/gift-card/{giftCardId}`

### Example

```bash
curl --location "$BASE_URL/card-program/gift-card/{giftCardId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
