---
title: 'List Wallet Gift Cards'
api: 'GET /card-program/gift-card'
---

List all the wallet gift cards linked to your account, along with their details.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Request

Endpoint: `GET /card-program/gift-card`

### Example

```bash
curl --location "$BASE_URL/card-program/gift-card" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
