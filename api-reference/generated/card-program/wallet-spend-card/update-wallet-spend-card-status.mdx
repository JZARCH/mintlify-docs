---
title: 'Update Wallet Spend Card Status'
api: 'POST /card-program/card/{cardId}/change-status'
---

Use this endpoint to update the status of a wallet spend card.

### **Path Parameters**

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `cardId` | `string` | ✅ | Unique identifier of the card whose status is to be changed. |

### **Request Body Parameters**

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `status` | `string` | ✅ | New card status. Possible Accepdted values: `active` and `inactive` |

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="cardId" type="string" required>
  Path parameter `cardId`.
</ParamField>

## Body parameters

<ParamField body="status" type="string">
  Request body field `status`.
</ParamField>

## Request

Endpoint: `POST /card-program/card/{cardId}/change-status`

### Example

```bash
curl --location "$BASE_URL/card-program/card/{cardId}/change-status" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
