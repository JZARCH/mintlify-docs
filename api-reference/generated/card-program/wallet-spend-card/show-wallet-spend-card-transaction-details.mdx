---
title: 'Show Wallet Spend Card Transaction Details'
api: 'GET /card-program/{cardProgramId}/transactions'
---

This API endpoint retrieves the transaction history and details for a specific spend card linked to a wallet using the card ID.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="cardProgramId" type="string" required>
  Path parameter `cardProgramId`.
</ParamField>

## Request

Endpoint: `GET /card-program/{cardProgramId}/transactions`

### Example

```bash
curl --location "$BASE_URL/card-program/{cardProgramId}/transactions" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
