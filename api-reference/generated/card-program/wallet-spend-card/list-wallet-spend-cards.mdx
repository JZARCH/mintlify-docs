---
title: 'List Wallet Spend Cards'
api: 'GET /card-program/card'
authMethod: 'bearer'
---

List all the wallet spend cards linked to your account, along with their details.

## Body parameters

<ParamField body="success" type="boolean">
  Request body field `success`.
</ParamField>

<ParamField body="meta" type="object">
  Request body field `meta`.
</ParamField>

<ParamField body="meta.total" type="number">
  Request body field `meta.total`.
</ParamField>

<ParamField body="meta.perPage" type="number">
  Request body field `meta.perPage`.
</ParamField>

<ParamField body="meta.currentPage" type="number">
  Request body field `meta.currentPage`.
</ParamField>

<ParamField body="meta.lastPage" type="number">
  Request body field `meta.lastPage`.
</ParamField>

<ParamField body="data" type="object[]">
  Request body field `data`.
</ParamField>

## Request

Endpoint: `GET /card-program/card`

### Example

```bash
curl --location "$BASE_URL/card-program/card" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
