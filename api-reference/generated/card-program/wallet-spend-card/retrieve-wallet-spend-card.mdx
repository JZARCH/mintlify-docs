---
title: 'Retrieve Wallet Spend Card'
api: 'GET /card-program/card/{cardId}'
---

Retrieves the details of a specific spend card using its unique ID.

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="cardId" type="string" required>
  Path parameter `cardId`.
</ParamField>

## Request

Endpoint: `GET /card-program/card/{cardId}`

### Example

```bash
curl --location "$BASE_URL/card-program/card/{cardId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
