title: 'Update a custom from address'
api: 'PUT /customFromAddresses/{customfromaddressId}'
authMethod: 'bearer'

Update details of an existing custom from address

Path parameters

 
  customFromAddressId
  Required    
         The id of the Custom from address to be update.
  




Payload parameters

 
  name
  Required
     String  
     Must be no longer than 50 characters.
  



  companyName
  Optional    
  String
   Must be no longer than 50 characters.
  



  addressLine1
  Required
  String
   Must be no longer than 100 characters.
  



  addressLine2
  Required    
  String
   Must be no longer than 100 characters.
  



  city
  Required  
  String
  Must be no longer than 100 characters.
  



  state
  Required   
  String
   Must be a 2 letter state short-name code 
  



  zip
  Required 
  String
   Must be no longer than 15 characters.
  



  phone
  Optional     
  String
   Must be no longer than 15 characters. 
  




  email
  Optional   
  String
   Must be no longer than 15 characters.
  



  note
  Optional   
  String
   An internal note that identifies this address,Must be no longer than 200 characters.
