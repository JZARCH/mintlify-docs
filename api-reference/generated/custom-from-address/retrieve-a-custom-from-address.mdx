---
title: 'Retrieve a custom from address'
api: 'GET /customFromAddresses/{customFromAddressId}'
---

Details of custom from address.

Path parameters

 
  customFromAddressId
  Required    
         The id of the Custom from address to be retrieved.
  



## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="customFromAddressId" type="string" required>
  Path parameter `customFromAddressId`.
</ParamField>

## Request

Endpoint: `GET /customFromAddresses/{customFromAddressId}`

### Example

```bash
curl --location "$BASE_URL/customFromAddresses/{customFromAddressId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
