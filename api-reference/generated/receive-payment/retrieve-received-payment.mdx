---
title: 'Retrieve Received Payment'
api: 'GET /receive-payments/{receivedPaymentId}/info'
---

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="receivedPaymentId" type="string" required>
  Path parameter `receivedPaymentId`.
</ParamField>

## Request

Endpoint: `GET /receive-payments/{receivedPaymentId}/info`

### Example

```bash
curl --location "$BASE_URL/receive-payments/{receivedPaymentId}/info" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
