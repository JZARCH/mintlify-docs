---
title: 'Retrieve Single Payment'
api: 'GET /getpaid-by-form/check/{referenceId}'
---

You can retrieve a single payment link check using `referenceId`

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="referenceId" type="string" required>
  Path parameter `referenceId`.
</ParamField>

## Request

Endpoint: `GET /getpaid-by-form/check/{referenceId}`

### Example

```bash
curl --location "$BASE_URL/getpaid-by-form/check/{referenceId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
