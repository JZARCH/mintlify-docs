---
title: 'Retrieve all Categories'
api: 'GET /categories'
authMethod: 'bearer'
---

List of all category records.


Query parameters

 
  perPage
     Number of records that you want to receive in the response,By default 10 records will be 
          listed
   
  
  page
    For navigating through pages ,By default first page will be listed 
  
  
    term
     Filter records by using any keywords
   



## Body parameters

<ParamField body="success" type="boolean">
  Request body field `success`.
</ParamField>

<ParamField body="data" type="object">
  Request body field `data`.
</ParamField>

<ParamField body="data.meta" type="object">
  Request body field `data.meta`.
</ParamField>

<ParamField body="data.meta.total" type="number">
  Request body field `data.meta.total`.
</ParamField>

<ParamField body="data.categories" type="object[]">
  Request body field `data.categories`.
</ParamField>

## Request

Endpoint: `GET /categories`

### Example

```bash
curl --location "$BASE_URL/categories" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
