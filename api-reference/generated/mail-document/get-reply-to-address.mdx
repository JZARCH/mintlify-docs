title: 'Get Reply To Address'
api: 'GET /documentmailing/getreplyto'
---

## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Query parameters

<ParamField query="perPage" type="string">
  The default value is 10, with a maximum of 100.
</ParamField>

<ParamField query="terms" type="string">
  We will search using these terms or keywords within the name, company name, phone, and email. If there's a match, you will receive results. This helps narrow down your search.
</ParamField>

## Request

Endpoint: `GET /documentmailing/getreplyto`

### Example

```bash
curl --location "$BASE_URL/documentmailing/getreplyto" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
