Details of custom to address.

Path parameters

 
  customToAddressId
  Required    
         The id of the Custom to address to be retrieve.
  



## Authorization

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer {{AUTH_TOKEN}}`.
</ParamField>

## Path parameters

<ParamField path="customToAddressId" type="string" required>
  Path parameter `customToAddressId`.
</ParamField>

## Request

Endpoint: `GET /customToAddresses/{customToAddressId}`

### Example

```bash
curl --location "$BASE_URL/customToAddresses/{customToAddressId}" \
  --header "Authorization: Bearer $AUTH_TOKEN" \
  --header "Accept: application/json"
```
