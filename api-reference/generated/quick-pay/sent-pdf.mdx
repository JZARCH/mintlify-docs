---
title: 'Sent Pdf'
api: 'POST /quickpay/mailpdf'
authMethod: 'bearer'
---

This is not a payment method. Sometimes you may want to send a QuickPDF to your employee, and this feature allows you to do that.

We also support an HTML to PDF document mailing feature. If you need a more advanced way of sending documents, consider using our Mail Document API: [Mail Document API](https://apiv3.onlinecheckwriter.com/#366c6da3-4691-4233-9f3e-3b02da2817d2).

For more examples, visit: [How to Send PDF to Your Recipient Easily](https://support.onlinecheckwriter.com/portal/en/kb/articles/how-to-sent-pdf-to-your-recipient-easily).

Shipping Type ID: 1, for regular USPS First Class mailing
