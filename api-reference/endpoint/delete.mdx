---
title: 'Create mail check'
api: 'POST /quickpay/mailcheck'
---

Create a physical mail check that Online Check Writer prints and mails to the recipient.

## Request

### Body

<ParamField body="source.accountType" type="string" required>
  Funding account type. For example, `"bankaccount"`.
</ParamField>

<ParamField body="source.accountId" type="string" required>
  ID of the funding bank account. This corresponds to `{{bankAccountId}}` in the Postman collection.
</ParamField>

<ParamField body="destination.name" type="string" required>
  Recipient name as printed on the mailed check.
</ParamField>

<ParamField body="destination.company" type="string">
  Optional company name.
</ParamField>

<ParamField body="destination.address1" type="string" required>
  Street address line 1 for the recipient.
</ParamField>

<ParamField body="destination.address2" type="string">
  Optional address line 2 or apartment/suite.
</ParamField>

<ParamField body="destination.city" type="string" required>
  City for the mailing address.
</ParamField>

<ParamField body="destination.state" type="string" required>
  State code. Use 2-character codes like `"TX"`.
</ParamField>

<ParamField body="destination.zip" type="string" required>
  Postal or ZIP code.
</ParamField>

<ParamField body="destination.phone" type="string">
  Optional phone number for the recipient.
</ParamField>

<ParamField body="destination.email" type="string">
  Optional email address for notifications.
</ParamField>

<ParamField body="destination.shippingTypeId" type="number" required>
  Shipping method identifier. For example: `1` (First Class), `3` (First Class with Tracking), `4` (Priority Mail), `5` (Express Mail), `12` (Standard Overnight FedEx).
</ParamField>

<ParamField body="payment_details.amount" type="number" required>
  Check amount.
</ParamField>

<ParamField body="payment_details.memo" type="string">
  Optional memo printed on the check.
</ParamField>

<ParamField body="payment_details.note" type="string">
  Optional internal note.
</ParamField>

<ParamField body="payment_details.issueDate" type="string">
  Optional issue date in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="payment_details.referenceId" type="string">
  Optional reference identifier from your system.
</ParamField>

## Response

On success, the API returns a JSON object indicating success and the created mail check identifier.

```json
{
  "success": true,
  "data": {
    "checkId": "exampleMailCheckId"
  }
}
```
