To simplify development, Zil Money provides dedicated Sandbox and Production environments.

## Sandbox

1. Create a test account.
2. Get API key from the developer panel:
   - `https://sandbox.onlinecheckwriter.com/manage/developer/index`
3. You can also review sandbox environment guidance here:
   - `https://developer.zilmoney.com/environments#sandbox`
4. Use this base URL in Postman or your client:

```bash
https://test.onlinecheckwriter.com/api/v3
```

## Production

Before moving to production, contact customer support to activate API access in the live environment.

1. Get production API key from:
   - `https://live.onlinecheckwriter.com/manage/developer/index`
2. Use this production base URL:

```bash
https://app.onlinecheckwriter.com/api/v3
```
