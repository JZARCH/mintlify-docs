To simplify development, Zil Money offers dedicated Sandbox and Production environments.

## Sandbox

1. Create a Test Account -> <a href="https://test.onlinecheckwriter.com/" target="_blank" rel="noopener noreferrer"><strong>Test Account</strong></a>
2. Get API Key -> <a href="https://sandbox.onlinecheckwriter.com/manage/developer/index" target="_blank" rel="noopener noreferrer"><strong>Developer Panel</strong></a>
3. Postman Setup -> Download Postman and set the base URL to:

```bash
https://test.onlinecheckwriter.com/api/v3
```

## Production

Before moving to production, please consult with our customer support team to activate the API in the production environment.

For assistance at any stage, contact our Live Chat Support -> <a href="https://onlinecheckwriter.com/contact/" target="_blank" rel="noopener noreferrer"><strong>Live Chat</strong></a>

1. Create a Production Account -> <a href="https://live.onlinecheckwriter.com/login" target="_blank" rel="noopener noreferrer"><strong>Production Account</strong></a>
2. Get API Key -> <a href="https://live.onlinecheckwriter.com/manage/developer/index" target="_blank" rel="noopener noreferrer"><strong>Production Developer Panel</strong></a>
3. Postman Setup -> Set the base URL to:

```bash
https://app.onlinecheckwriter.com/api/v3
```
